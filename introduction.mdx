## Features

VisionFly API offers a comprehensive set of image transformation options:

- **Resize** - Change image dimensions while maintaining aspect ratio
- **Format Conversion** - Convert between formats (JPG, PNG, WebP)
- **Quality Adjustments** - Control compression quality to optimize file size
- **Cropping** - Precise or smart cropping of images
- **Visual Adjustments** - Modify brightness, contrast, and saturation
- **More to Come** - Regular updates with new features based on developer feedback

## Getting Started

To start using VisionFly API, you'll need:

1. An API key (sign up at [dashboard.visionfly.ai](https://dashboard.visionfly.ai))
2. Understanding of the basic URL structure
3. Knowledge of available transformation parameters

Continue to the [Quick Start](/api-reference/quick-start) guide to send your first request, or explore our [API Reference](/api-reference/endpoints) for detailed information.

```javascript
// Example: Resizing an image
fetch(
  "https://api.visionfly.ai/v1/transform/w800-h600-webp/example.com/image.jpg",
  {
    headers: {
      "VF-API-Key": "your-api-key",
    },
  }
)
  .then((response) => response.blob())
  .then((imageBlob) => {
    const imageUrl = URL.createObjectURL(imageBlob);
    document.getElementById("result").src = imageUrl;
  });
```
